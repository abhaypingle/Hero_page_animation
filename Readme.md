-->ORBIT IMAGE ANIMATION

A simple web animation project where an image is animated using pure CSS. 
The image moves smoothly in an orbit-like path with scaling effects to create a modern landing page style animation.

-->FEATURES
- Smooth CSS keyframe animation
- Orbit movement effect
- Image scaling during movement
- No JavaScript required
- Fully built using HTML and CSS
-->HOW TO RUN
1. Download or clone this repository.

2. Open the project folder.

3. Open the index.html file in your browser.

That's it! The animation will start automatically.
-->LEARNING OUTCOME

Through this project, I learned how to use CSS @keyframes, transform, translate(), and scale() to create smooth animations without using JavaScript.
